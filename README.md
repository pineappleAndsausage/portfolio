# 박노열 포트폴리오

데이터와 제품을 연결하는 시스템을 만드는 Senior Software Engineer 박노열의 포트폴리오 사이트입니다.

## 프로젝트

| 프로젝트 | 분야 | 핵심 내용 |
| --- | --- | --- |
| [PlusInsight](case-studies/plusinsight.html) | Data Platform | 온프레미스 환경에서 실시간·일별 분석, 다고객 배포와 관측을 운영하는 공간 행동 분석 플랫폼 |
| [Samramansang](case-studies/samramansang.html) | Retrieval Systems | 시간적 움직임을 임베딩·검색해 실시간 상호작용 경험으로 연결한 시스템 |
| [Lumenar Series](case-studies/lumenar.html) | Interactive Systems | 관객의 입력을 제어 가능한 물리적 빛의 출력으로 변환하는 인터랙티브 시스템 |
| [Orbital Cells](case-studies/orbital-cells.html) | Real-time Graphics | 하나의 시간 기반 생명주기 모델을 웹·데스크톱·임베디드 런타임에서 재현하는 생성형 작업 |
| [Nachos](case-studies/nachos.html) | Data Operations | 고객별 요구와 공통 수집·품질관리 흐름을 분리한 AI 데이터 수집 플랫폼 |

HTML, CSS, JavaScript를 기반으로 제작했으며, 프로젝트별로 Python, TypeScript, React, Next.js, PostgreSQL, AWS, Kafka, ClickHouse, OpenSearch, PyTorch, WebGL, C++/OpenGL 등을 다룹니다.
